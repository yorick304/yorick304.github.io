vue颜色选择器

  • javascript

ANALYZE

在学习vue的过程中小试牛刀，顺便总结在这个过程中的收获和遇到的坑。在学习过程中转换了原来使用angularjs的思维来进行编写代码，组件的概念深入人心，页面由组件构成。
在编写vueColorPicker的过程中，组件减少多外部的依赖，而是专注于完成指定的功能，这个功能独立单一，便于提炼为了让组件更加具有灵活性和可配置性，需要向外提供更规范的props。

  • 特点
    • 用以激活HTML5颜色面板
    • 颜色显示小方块
    • 颜色色盘
    • 标准颜色
    • 向外广播选择的颜色，方便父级元素的接收
